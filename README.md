# docs-elapas

Documentación del proyecto académico **Sistema de Gestión de Recaudaciones y Cortes para ELAPAS - Sucre**.

## Integrantes
- Franz Caballero
- Gadiel Cordova
- Grover Diaz

## Objetivo
Organizar la documentación de planificación, análisis, arquitectura, seguimiento, pruebas y presentación del MVP.

## Repositorios del proyecto
- `backend-elapas`: backend del sistema
- `movil-elapas`: interfaz movil del MVP
- `docs-elapas`: documentación y gestión del proyecto

## Estructura
- `01-planificacion/`: alcance, metodología, cronograma y organización
- `02-requerimientos/`: requerimientos e historias de usuario
- `03-analisis/`: actores, factibilidad, viabilidad y riesgos
- `04-arquitectura/`: módulos, flujos, decisiones técnicas y modelo de datos
- `05-seguimiento/`: actas, avances y decisiones
- `06-pruebas/`: casos y resultados de prueba
- `07-presentacion/`: guion, resumen y evidencias
- `08-anexos/`: referencias y recursos

## Estado actual
En planificación y definición del MVP.
